# Union WELL Web Project

A portfolio-ready PHP/MySQL website inspired by student union and campus wellness web systems.

## Tech Stack
- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- XML/RSS
- Git
- Apache via XAMPP or Laragon

## Features
- Responsive multi-page public website
- Dynamic events and class schedule from MySQL
- Admin login and content management
- Contact form storing submissions in the database
- RSS feed for events
